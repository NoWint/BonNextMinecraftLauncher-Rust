Features

Everything you need.
Nothing you don't.

From authentication to mod management, BonNext provides a complete launcher experience with performance to spare.

⚡
High-Performance Downloads
Parallel multi-threaded downloads with SHA1 integrity verification and automatic mirror failover across Official, BMCLAPI, and MCBBS sources.
🔒
Dual Authentication
Microsoft OAuth 2.0 device-code flow with automatic token refresh, plus offline mode with deterministic UUIDs. Multi-account switching.
📦
Smart Instance Management
Create, configure, clone, and delete isolated instances. Each has its own mods, saves, and configs — sharing core libraries via hard links.
🧪
Dual Mod Browser
Browse, search, and install from both Modrinth and CurseForge. Filter by version and loader. Gallery and list view modes.
New in SwiftUI Shell
🎨
Liquid Glass Design
Native macOS 26 Liquid Glass effect with CSS blur fallback on other platforms. Apple HIG-aligned design tokens, SF Pro typography, and spring animations. A launcher that feels at home on every platform.
/* Liquid Glass — Two-Tier Strategy */
/* 1. macOS 26+: NSGlassEffectView */
/* 2. Default: CSS backdrop-filter */

.liquid-glass-sidebar {
  background: var(--swift-glass-tint);
  backdrop-filter: blur(40px) saturate(180%);
}

/* Native glass overrides the CSS blur */
.glass-native-liquid .liquid-glass-sidebar {
  backdrop-filter: none;

Architecture

Rust at the core.
React at the surface.

Tauri v2 bridges the Rust backend and React frontend for a lightweight, secure, and performant desktop experience.

Frontend — React 18 + TypeScript Components, Pages, Stores, CSS Modules
Bridge — Tauri v2 IPC invoke() ↔ #[tauri::command]
Backend — Rust + Tokio Auth, Download, Launch, Version, Loader, Instance

Data Flow

1

User Action

Click, search, install — React component captures the event.

2

Tauri IPC

api.ts invoke() sends typed parameters to the Rust backend.

3

Rust Processing

Async command handler processes auth, download, launch, and more.

4

Real-Time Updates

app.emit() streams progress events back to the React frontend.

Build from Source

Requires Rust 1.80+, Node.js 20+, and pnpm 9+

# Clone and build
git clone https://github.com/bonnext/bonnext.git
cd bonnext

pnpm install        # Install frontend dependencies
pnpm tauri dev    # Run in development mode
pnpm tauri build  # Production build
