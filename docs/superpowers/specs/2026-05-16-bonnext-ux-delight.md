## 核心设计原则

1. **零帧延迟** — 任何交互都必须在 100ms 内给出视觉反馈
2. **物理真实** — 所有动画使用 ease-out-expo / spring 曲线，拒绝 linear
3. **Minecraft 文化** — 文案、音效、视觉都从 Minecraft 世界汲取灵感
4. **渐进式复杂** — 新用户看到的是极简，高级用户能发现深度功能
5. **处处有惊喜** — 每个角落都可能藏着一个小彩蛋或巧思
