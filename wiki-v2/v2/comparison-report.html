3.2 关键技术决策

动画优化

grid-template-rows 替代 max-height

使用 CSS Grid 的 grid-template-rows 动画实现折叠效果，比 max-height 更流畅，无需预设高度值。

状态管理

localStorage 键值设计

使用文件夹路径（如"概述 / 项目简介"）作为 key，确保状态在文件结构变化时仍能正确恢复。

搜索性能

防抖 + 批量 DOM 更新

150ms 防抖减少过滤频率；使用 DocumentFragment 批量更新 DOM，避免多次重排。

无障碍

prefers-reduced-motion

尊重用户系统设置，在减少动画偏好下禁用所有过渡效果，确保可访问性。
